# Trial Webpage

A simple React-based webpage deployed via GitHub Pages.

## Live Demo
https://shivaprasadacharya.github.io/trialwebpage/

## Development1
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
